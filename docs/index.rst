Introduction
------------

This package implements the convolutional omics kernel network introduced by Jonas C. Ditz et al., "Convolutional Kernel Networks for Interpretable End-to-End Learning on (Multi-)Omics Data", 2022.


Requirements
------------

*   Python >= 3.7
*   PyTorch == 1.8.1
*   NumPy == 1.19.2
*   SciPy == 1.6.1
*   Matplotlib == 3.3.4
*   Pandas == 1.2.3
*   SciKit-Learn == 0.24.1
*   BioPython == 1.78
